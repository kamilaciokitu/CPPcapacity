CPP End-to-End Planning Workflow

Datasets

Step 2 — Creating a New Dataset
"
Ensuring your data is accurate, complete, and relevant is crucial because the platform's analysis, modelling, and final recommendations all depend on the datasets you upload.

Data templates are provided.
Start a New Dataset
In CPP go to Datasets in the left menu → click the + button → fill in the form.
Field 1
Dataset Name
e.g. "Benin – Cell Towers"
Field 2
Country
Select from ITU member states list
Field 3
Dataset Type
Select type → download template
Then: Select file(s) → Click Validate and Create

CPP validates the file against the template schema before creating the dataset. Fix any errors flagged before proceeding to the next step.

Step 3 — Scenarios & Project Setup
Creating a New Project — 4 Steps

In CPP, go to Projects → Add New. You may Save or Cancel at any step — once saved, the project appears as a Draft in the Project List.

Step 1 — Project Information
Enter: Project Name, Country of Focus (ITU member states list), Project Description
Step 2 — Scenario & Model
Select: Maximize Net Present Value or Minimize Total Cost of Ownership. Enter parameter values for each scenario parameter.
Step 3 — Model Parameters & Datasets
Select datasets (Cellular Towers, Transmission Nodes, POIs, Mobile Coverage). Ensure they match the country and model requirements.
Step 4 — Run Calculations
Click Start Calculation. Can take several minutes to hours. Monitor status in the Project List — shows as calculated when complete.

Step 4 — Models

Models provide a structured setup for planning while simulating specific telecommunication technologies. In Step 2 of creating a project, click + Add a model to your scenario. Enter parameter values for each selected model.

Step 6 — Use Insights for Planning & Policy
"
The CPP provides a deep and practical understanding of your country's connectivity landscape.
The results can be used to
Institutional Connectivity

Identify which towers or transmission nodes can serve specific schools, hospitals, and clinics. Prioritise institutions by population served and feasibility.

Rural Expansion Planning

Compare fibre, microwave, and cellular solutions for remote areas. Plan fibre expansion or cellular upgrades based on CPP cost modelling.

Redundancy and Resilience

Explore satellite or microwave backup options for critical institutions. Design resilient networks that maintain connectivity under failure scenarios.

National Strategy Development

Use CPP outputs to shape broadband plans, inform national broadband strategies, and guide investment and budgeting discussions with development partners.
